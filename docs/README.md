# docs/

Free-form notes about the project.

## Files planned for this folder

| File | Purpose |
|------|---------|
| `meeting_notes.md` | One section per team meeting, dated |
| `decisions/<topic>.md` | One file per major design decision (e.g., `coupling_choice.md`, `node_count.md`) |
| `todo.md` | Optional shared todo list |

## What goes here

- Meeting summaries
- Design discussions and the decisions made
- Open questions for the team
- Anything that helps future team members understand why something is the way it is

## What does NOT belong here

- Code (goes in `src/` or `scripts/`)
- Slide content (goes in `slides/`)
