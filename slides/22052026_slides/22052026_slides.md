# Cortical EEG slowing in resting-state Parkinson's disease

<div class="subtitle">Progress update: hypothesis, model, and next steps</div>

<div class="meta">PD EEG group · 22.05.2026</div>

---

# Hypothesis (refined)

## What we predict

> Compared to age-matched healthy controls, Parkinson's disease patients at rest show:
> - **slowing of the alpha peak frequency** (8–13 Hz)
> - **increased theta-band power** (4–8 Hz)
> - **decreased cortical beta-band power** (13–30 Hz)

## Refinement based on data

Originally formulated as *"decreased alpha power"*. Group comparison on 149 subjects showed that **alpha relative power is essentially unchanged** between groups, while the **alpha peak frequency drops by ~1 Hz in PD** (median 9.25 → 8.00 Hz, p < 0.001).
The wording was refined to *"alpha peak slowing"* to reflect the actual change in the data.

# Model: Wilson-Cowan network

## Why Wilson-Cowan

- **Biologically interpretable**: each node is an excitatory–inhibitory neural population pair, a textbook minimal model of cortex.
- **Single parameter maps directly to the hypothesis**: the inhibitory time constant τ_inh controls oscillation speed, which is precisely what our refined hypothesis predicts will change.
- **Available in the course toolkit**: neurolib provides a ready Wilson-Cowan implementation, so the team focuses on the modelling decisions rather than reimplementing equations.

## Configuration

![bg right:42% w:430](figures/wc_network.svg)

- **N = 8 nodes**, all-to-all coupling with global strength K_gl
- All nodes share a single global τ_inh (matches the whole-cortex slowing in our hypothesis)
- Deterministic limit-cycle regime (no noise) so the simulated peak frequency depends only on τ_inh
- Network output: one alpha peak whose frequency is set by τ_inh

# Status and next steps

## Already done

- **Feature extraction** across 149 participants → per-subject alpha peak frequency, alpha / theta / beta relative power.
- **Data-level group comparison**: all three predicted directions are significant (p < 0.001 for alpha peak, theta power, beta power).
- **Wilson-Cowan model implemented** and validated against single-node and N=8 cases.
- **τ_inh → peak frequency lookup table** generated by sweeping τ_inh from 11 to 32 ms.

## Remaining work before final report

- **Per-subject fitting**: invert the lookup curve to obtain each participant's personal τ_inh from their measured alpha peak.
- **Model-level group comparison**: test whether the PD τ_inh distribution differs from the Control τ_inh distribution.
- Translate the final result into the assignment's "+Δτ_inh" statement.

# Open questions for the podium

## On the data

- **Alpha peak slows without a change in relative alpha power.** Is this what we should expect at this cohort's PD severity, or does it reflect the choice of relative (vs absolute) band power? Worth checking absolute power before claiming this is a general pattern.

## On the model

- **Single global τ_inh** captures the whole-cortex slowing assumption. Could a **per-node τ_inh** (regional heterogeneity) reveal something the global model misses, even if the data is preprocessed at one spatial scale?
- **Cortical beta decrease is not directly modeled** by Wilson-Cowan with a single τ_inh; it is only consistent with the general slowing pattern. Should we add a second oscillator to model beta explicitly, or accept that as a known limitation?

## On methodology

- We did not generate synthetic data. Would a small synthetic dataset (model-generated EEG-like signals) help validate the per-subject fitting procedure before applying it to real data?
